  ==Node-Based Data Structures==
  ==Linked Lists==
  >Data structure that represents list of items

In addition to the data stored within the node, each node also stores the memory address of the next node in the linked list

Advantage of link list over array is that program doesn't need to find a bunch of empty memory cells in a row to store its data. Instead the program can store the data across many cells that are not necessarily adjacent to each other

One case where linked lists shine is when we examine a single list and delete many elements from it

==Doubly Linked Lists==
> each node has two links, one that point to the next node, and one that pints to the preceding node
> also keeps track of both first and last nodes
