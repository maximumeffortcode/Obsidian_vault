==Dealing With Space Constraints==
[Space Complexity]- how much memory an algorithm consumes

==Big 0 Notation as Applied to Space Complexity==
Big 0 can be used to describe how much space an algorithm takes up: 
	for N elements of data, an algorithm consumes a relative number of additional data elements in memory

0(1) by space complexity means that the memory consumed by an algorithm is constant no matter how large the data

Judge space complexity based on additional memory consumed aka auxiliary space

Don't count the original data
