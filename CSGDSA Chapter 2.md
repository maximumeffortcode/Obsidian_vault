==Why Algorithms Matter==
[Algorithm]- particular process for solving a problem when applied to computing, algorithm refers to a process for going about a particular operation
-multiple algorithms exist that can achieve a particular operation

==Ordered Arrays==
only difference to array is with ordered arrays require that the values are always kept in order

==Searching An Ordered Array==
- can stop a search early even if value is not contained within the array
- because an ordered array needs to be in order once value is passed no need to search any further
- big advantage of ordered array over regular array is that ordered array allows for binary search and is much faster algorithm than linear search

==Binary Search==
"guess in the middle then go from there"

==Binary Search vs Linear Search==
- with an array containing 100 values, here are the maximum numbers of steps it would take for each type of search:
	- [Linear Search]: 100 steps
	- [Binary Search]: 7 steps
- using an ordered array, slower insertion but much faster search
